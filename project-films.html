Mes Films Préférés

Application web académique, BTS SIO

Description du projet

Mes Films Préférés est une application web permettant de découvrir et gérer ses films préférés grâce à une API externe (comme TMDB). L'utilisateur peut rechercher des films, consulter les détails (synopsis, acteurs, notes), créer des listes personnelles et partager ses découvertes. L'interface est responsive avec un système d'authentification utilisateur complet.

Technologies utilisées

HTML CSS JavaScript API REST Responsive Design Authentification
