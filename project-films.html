Mes Films Préférés

Application web de recherche et gestion de films via API

Description du projet

Mes Films Préférés est une application web développée en PHP dans le framework Laravel qui exploite une API REST externe (The Movie Database - TMDb) pour permettre aux utilisateurs de découvrir, rechercher et gérer leurs films préférés. L'application inclut un système d'authentification, d'amis, de partage, une recherche avancée, la gestion des favoris et un système de notation personnelle.
