CastleFightArena est un jeu de combat développé en Java / JavaFX.
Le joueur choisit un personnage parmi 4 disponibles, puis lance un combat contre un autre personnage.

Le jeu fonctionne en tour par tour : chaque joueur attaque à son tour en cliquant sur un bouton.

Fonctionnement
Le joueur sélectionne un personnage
Il clique sur "Fight" pour lancer l’attaque
Les attaques s’alternent automatiquement
Les dégâts sont aléatoires
À chaque attaque :
une animation est jouée
la fenêtre tremble
la barre de vie diminue
Quand un personnage n’a plus de vie :
une animation de victoire/défaite s’affiche
la partie peut recommencer
Base de données

Les scores sont enregistrés en base de données, ce qui permet de garder un historique des résultats.

Technologies
Java
JavaFX
Base de données (SQL)
Objectif

Ce projet m’a permis de pratiquer :

les interfaces graphiques avec JavaFX
les animations
la logique de jeu simple
la connexion à une base de données
