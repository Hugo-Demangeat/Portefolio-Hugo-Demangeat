Script API OVH

Automatisation de gestion de domaines - Décembre 2025

Description du projet

Développement d'un script Python utilisant l'API OVH pour automatiser la gestion des domaines. Le script permet de récupérer les informations des domaines, gérer les serveurs DNS, automatiser les renouvellements et générer des rapports détaillés. Intégration avec les services OVH Telecom pour une gestion complète de l'infrastructure web.
