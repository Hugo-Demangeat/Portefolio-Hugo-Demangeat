Go FoodTruck Manager

Application mobile académique, BTS SIO

Description du projet

Go FoodTruck Manager est une application mobile de gestion des commandes pour food truck développée avec WinDev. Elle permet de gérer le catalogue de produits, prendre des commandes, suivre les ventes en temps réel et générer des rapports. L'interface tactile est optimisée pour une utilisation en mobilité avec synchronisation des données.
